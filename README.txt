GuessingGame App
----------------
Compile: javac *.java
Usage (Server): java GameServer
Usage (Client): telnet localhost:6001
Description: A yes/no multiplayer celebrity guessing game.

ToyFTPServer App
----------------
Compile (Server): javac *.java
Compile (Client): javac ToyFTClient.java
Usage (Server): java ToyFTServer [place ANY file(s) in this subfolder]
Usage (Client): java ToyFTClient [when requesting files, also inlude the file extension]
Description: Transfer any files from server to client.
